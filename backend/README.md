# Hue Travel - Backend

Đây là phần máy chủ (backend) của dự án **Hue Travel**, được xây dựng bằng Node.js và Express.

## 📦 Công nghệ sử dụng

- Node.js
- Express.js
- MongoDB / MySQL (tùy vào CSDL bạn dùng)
- Dotenv
- Nodemon (dev)
- JWT (nếu có xác thực)
- Mongoose / Sequelize (ORM)

## 🚀 Cài đặt và chạy

```bash
# Cài đặt thư viện
npm install

# Chạy ở môi trường phát triển
npm run dev

# Hoặc chạy production
npm start
